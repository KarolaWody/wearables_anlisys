# Workflow Rules

## Always Plan First
Before any code changes, present a clear plan with numbered steps.

## Show All Steps
Lay out every action that will be taken — file creations, edits, commands, etc.

## Ask for Permission
Do NOT execute any step until the user explicitly approves the plan. After approval, execute step by step, confirming after each if needed.

## This File is First
This `AGENTS.md` is the first file read in every session. Follow its rules strictly.
